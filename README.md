# Animora — animoracare.com

**Privacy-first elderly care with Edge AI.**
No cameras. No wearables. Just peace of mind.

## About

Animora monitors elderly individuals living alone using invisible Zigbee sensors and local Edge AI (Raspberry Pi). The system learns daily routines in 14 days and alerts family members when anomalies are detected — all without cameras, microphones, or wearables.

## Tech Stack

- Raspberry Pi 4/5 (Edge AI Hub)
- Zigbee sensor network (door contacts, motion, vibration)
- Home Assistant Core
- Bayesian Beta-Binomial routine learning
- Docker containerized deployment
